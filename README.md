# ToDo List Module for Odoo

An Odoo module designed to streamline task management, categorize activities with tags, and ensure process integrity through sub-task validation.

## Installation & Setup

1.  Copy the module folder into your Odoo addons directory.
2.  Restart Odoo.
2.  Update the App List in Odoo.
3.  Acitvate the module.

## How to Use

1.   Create a Todo List
2.   Click Progress to start
3.   Add tasks and mark them complete
4.   Add attendees if needed
5.   When all tasks are done, click Done

Use the menu to view All Todo List / Uncomplete Todo List / Complete Todo List.
